A [snap](https://snapcraft.io/about) package for [Open OnDemand](https://openondemand.org) —
an interactive web-based interface for seamlessly accessing your supercomputing
resources over the internet. Open OnDemand empowers students, researchers, and industry
professionals by connecting their powerful minds with raw computing power.

## ✨ Getting started

### Installing the Open OnDemand snap

The latest, stable version of the Open OnDemand snap can be installed using the following command:

```shell
sudo snap install ondemand
```

## 🛠️ Development

The project uses [snapcraft](https://snapcraft.io/docs) for building the snap:

```shell
snapcraft pack
```

If you're interested in contributing your work to the Open OnDemand snap,
take a look at our [contributing guidelines](./CONTRIBUTING.md) for further details.
